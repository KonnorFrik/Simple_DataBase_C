/ Simple C DB
/ author: KonnorFrik

This is simple CRUD for binary Database on C

For compile run 'make all' or 'make main'
but don't forget run 'make clean' after 'main target'

Compiled programm with name 'main' wait you in 'bin' forlder of This folder

For interact with db use the number from the menu that you will see in your terminal

*******************DUNGER PROGRAMMING ZONE*******************
* If you want change struct that stored in DB               *
* Write your own struct in 'struct.h' - for read            *
* Change 'fill_module' function in 'database.c' - for write *
* Change 'FOR CHANGES' sections in 'update.c' - for update  *
*************************************************************
